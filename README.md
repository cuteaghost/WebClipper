描述
-----------
这是一款Windows平台下支持Chrome浏览器的网页随摘系统。

本网页随摘系统分为三个部分:

- 本地客户端
- 通信模块
- Chrome扩展

使用说明
----------


1. 安装Chrome扩展
2. 修改nativemessaging/Debug/com.xiaowenwen.webclipper-win.json中的allowed_origins字段，改为扩展管理界面中的ID号
3. 运行nativemessaging/Debug/install_host.bat
4. 即可使用
